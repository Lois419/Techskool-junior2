MEASURE OF CENTERAL TENDENCY (Mode and Range)

Mode

The mode refers to the value that appears most frequently in a set of data.
To find the mode, you need to count how many times each value occurs in the data set and identify the value with the highest frequency.

Let's say you have the following data set: 2, 3, 4, 2, 5, 2, 6.
The number 2 appears 3 times, The number 3 appears 1 time, The number 4 appears 1 time, The number 5 appears 1 time, and The number 6 appears 1 time.
In this example, the mode is 2 because it appears more often than any other number in the data set.

Range

Range typically refers to the range of a set of data, which is the difference between the highest and lowest values in that set. It indicates how spread out the data is.

Let's say you have the following data set: 2, 3, 4, 2, 5, 2, 6.
The number 2 appears 3 times, The number 3 appears 1 time, The number 4 appears 1 time, The number 5 appears 1 time, and The number 6 appears 1 time.
In this example, the mode is 2 because it appears more often than any other number in the data set.

How to solve for range.

  • Order the data:

    Arrange the numbers in ascending (smallest to largest) or descending order.

  • Identify the highest and lowest values:

    Find the maximum and minimum values in the ordered data.

  • Calculate the difference:

    Subtract the lowest value from the highest value.
