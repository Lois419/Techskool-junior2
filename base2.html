Counting in Base 2

Introduction

A binary number is a number expressed in the base 2 or binary numeral system.
The binary system uses only two digits: 0 and 1.
Each position in a binary number represents a power of 2
The rightmost digit represents 2^0 (1).
The next digit to the left represents 2^1 (2), then 2^2 (4), and so on.

Counting in Base 2
